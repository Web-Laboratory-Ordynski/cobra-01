                a[data-filetype="image"] {
                    color: red;
                   }

X[foo~="bar"]

X[foo~="bar"]

CSS

Символ тильда (~) позволяет выбрать атрибут, в котором есть значения, разделённые пробелом.

                a[data-info~="external"] {
                    color: red;
                   }
                    
                   a[data-info~="image"] {
                    border: 1px solid black;
                   }

HTML

                <a> href="#" data-info="external image"> Нажми на меня, дурашка </a>
Нажми на меня, дурашка

X:checked

CSS

Данный псевдокласс выберет только отмеченные элементы пользовательского интерфейса: радиокнопки или чекбоксы. 

                input[type=radio]:checked {
                    border: 1px solid black;
                   }

HTML

                <form >
                    <p><b>Какое у вас состояние разума?</b></p>
                     <p><input name="dzen" type="radio" value="nedzen"><label for="dzen"> Не дзен</label> </p>
                     <p><input name="dzen" type="radio" value="dzen"> <label for="dzen"> Дзен</label> Дзен</p>
                     <p><input name="dzen" type="radio" value="pdzen" checked> <label for="dzen">Полный дзен</label> </p>
                     
                   </form>
